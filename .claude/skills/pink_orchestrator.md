# Pink Orchestrator (클라이언트 렌더링 지시서)

**Role**: `edit_instruction.json`을 기반으로 로컬 원본 고화질 소스들을 조합, `FFmpeg.wasm` 체계를 타겟으로 브라우저 내에서 최종 `.mp4` 렌더링.

**AI 에이전트 절대 준수 지침**:
1. 지시서의 `in/out` 타임스탬프 배열을 엄격히 추종하여 소스를 분할/병합할 것.
2. **[흐름 보존 최우선]**: 유저 원본 액션 구간이 템플릿 슬롯보다 길 경우, 임의로 영상을 자르지(Cut) 말 것. 슬롯 길이에 맞추기 위해 1순위로 가변 슬롯(비트 딜레이)을 적용하고, 불가능할 경우 2순위로 `speed` 변수를 적용하여 배속 렌더링(Speed-Ramp) 처리할 것.
3. **[Semantic Muting 강제]**:
   - `template.json` 내 해당 슬롯의 `"keep_audio"` 플래그가 `true`일 경우, 원본 영상의 오디오 트랙(음식 굽는 소리 등 현장음)을 필히 보존하여 믹싱할 것.
   - 플래그가 `false`일 경우, 해당 클립 렌더링 시 오디오 트랙을 전면 삭제(Mute)할 것.
4. **결과물 비디오 트랙에 BGM 삽입 절대 금지.** (BGM은 플랫폼 종속적이므로 사용자가 틱톡/인스타 플랫폼 내에서 사후 추가해야 함).
