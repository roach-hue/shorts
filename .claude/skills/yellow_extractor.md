# Yellow Pipeline (템플릿 추출 코어 지시서)

**Role**: 유튜브 쇼츠에서 편집 문법과 오디오 리듬을 추출하여 `template.json`을 생성.

**AI 에이전트 절대 준수 지침**:
1. `yt-dlp`를 사용하여 480p 해상도의 타겟 쇼츠를 임시 다운로드할 것. (60초 초과 영상은 즉각 Reject)
2. `librosa`를 사용하여 오디오의 비트 타임스탬프(ms 단위)와 드랍 포인트를 정확히 추출할 것.
3. `PySceneDetect`로 화면 전환점(컷)을 분할할 것.
4. 각 컷의 중간 지점에서 대표 키프레임 1장을 추출할 것.
5. VLM(`moondream2`)을 사용하여 시각적 속성(shot_type, objects, motion)을 필히 태깅할 것.
6. OCR을 수행하여 자막의 텍스트, 화면 내 위치, 그리고 크기 변동 애니메이션(`style`)을 추출할 것.
7. 반드시 구비된 스키마 규격(`docs/schemas/template.json`)에 일치하는 구조체를 출력할 것.
8. **[CRITICAL] 법적 저작권 방어를 위해 원본 영상 및 음원은 분석 종료 즉시 서버에서 영구 삭제할 것.** 1바이트도 저장 금지.
